# JK Tool 使用文档

## 1. 项目概述

JK Tool 是一个基于 Java 的工具库，提供了丰富的实用工具类，包括地理编码服务、坐标转换、HTTP 请求、日期时间处理、加密解密、系统工具等功能。该项目适用于需要集成各种实用工具的 Java 应用程序。

## 2. 核心功能模块

### 2.1 地理编码服务

#### 2.1.1 高德地图服务

- **反地理编码**：通过 `AmapRegeoCoder` 类实现，将经纬度转换为结构化地址信息。
- **地理编码**：通过 `AmapGeoCoder` 类实现，将地址信息转换为经纬度。
- **区域查询**：通过 `AmapDistrictQuery` 类实现，根据关键字查询区域信息。
- **坐标转换**：通过 `AmapCoordinateConverter` 类实现，通过API进行坐标转换。
- **POI搜索**：通过 `AmapPoiSearcher` 类实现，根据关键字、分类、区域等条件搜索兴趣点信息。

#### 2.1.2 天地图服务

- **地理编码与反地理编码**：通过 `TiandituGeocoder` 类实现，支持将经纬度转换为结构化地址信息、将地址信息转换为经纬度。
- **行政区域查询**：通过 `TiandituAdministrative` 类实现，支持查询行政区域信息。

### 2.2 坐标转换工具

- **本地坐标转换**：通过 `CoordinateConverter` 类实现，支持WGS-84、GCJ-02、BD-09坐标系之间的本地转换。
  - WGS-84转GCJ-02：将GPS原始坐标转换为火星坐标系
  - GCJ-02转WGS-84：将火星坐标还原为GPS原始坐标
  - GCJ-02转BD-09：将火星坐标（高德/腾讯地图）转换为百度坐标系
  - BD-09转GCJ-02：将百度坐标系转换为火星坐标系
  - WGS-84转BD-09：将GPS坐标直接转换为百度坐标系
  - BD-09转WGS-84：将百度坐标直接还原为GPS坐标

### 2.3 通用工具类

#### 2.3.1 HTTP 请求工具

- **GET 请求**：支持无参数、带参数、自定义字符集的 GET 请求。
- **POST 请求**：支持表单参数的 POST 请求。
- **POST JSON 请求**：支持发送 JSON 格式数据的 POST 请求。

#### 2.3.2 日期时间处理

- **日期格式化**：支持将 `LocalDate`、`LocalDateTime` 格式化为指定模式的字符串。
- **日期解析**：支持将字符串解析为 `LocalDate`、`LocalDateTime`。
- **日期转换**：支持 `Date` 与 `LocalDate`、`LocalDateTime` 之间的相互转换。
- **日期计算**：支持计算两个日期之间的天数、小时、分钟、秒数差。
- **日期操作**：支持添加天数、小时、分钟、秒数。
- **日期比较**：支持检查两个日期是否为同一天，检查日期是否为今天。
- **年龄计算**：根据生日计算年龄。
- **月份操作**：获取月份的第一天和最后一天。

#### 2.3.3 字符串处理

- **判空操作**：检查字符串是否为空、非空。
- **分割与连接**：分割字符串，连接字符串数组。
- **替换与格式化**：替换字符串中的内容，格式化字符串。

#### 2.3.4 文件操作

- **文件读写**：读取文件内容，写入内容到文件。
- **文件操作**：创建文件，删除文件，检查文件是否存在。
- **目录操作**：创建目录，遍历目录。

#### 2.3.5 配置文件处理

- **Properties 操作**：从类路径或文件系统读取、修改和保存 properties 配置文件。

#### 2.3.6 Bean 操作

- **Bean 转换**：支持 Bean 之间的转换。
- **Bean 属性操作**：提供 Bean 属性的读取和设置工具。
- **Bean 字段过滤**：支持灵活过滤 Java Bean 中的字段。

#### 2.3.7 其他通用工具

- **Freemarker 模板**：简化模板渲染和数据绑定操作。
- **JWT 令牌**：支持 JWT 令牌的生成、解析和验证功能。
- **注解工具**：提供注解相关的工具方法。
- **ID 生成**：提供日期 ID 和通用 ID 生成工具。

### 2.4 加密解密工具

- **安全工具类**：提供 AES/DES 对称加密、MD5/SHA 哈希算法、HMAC 消息认证码、UUID 生成等安全相关功能。
- **AES 加解密**：提供 AES 对称加密算法的完整实现，支持多种模式和填充方式。
- **RSA 加解密**：提供 RSA 非对称加密算法的密钥生成、加密、解密、签名和验证功能。
- **DSA 数字签名**：提供 DSA 数字签名算法的密钥生成、签名和验证功能。
- **摘要算法**：支持 MD5、SHA-1、SHA-256、SHA-512 等常用摘要算法。

### 2.5 系统工具类

- **Hosts 文件管理**：支持 Hosts 文件的读取、编辑和备份功能。
- **DNS 解析工具**：支持 DNS 查询和 DNS over HTTPS 功能。
- **IP 地址处理**：支持 IP 地址解析和代理 IP 识别。
- **进程管理**：支持跨平台的进程启动、查询和管理功能。
- **日志工具**：提供简洁的日志记录功能。

### 2.6 邮件工具

- **邮件发送**：支持发送邮件功能。
- **邮件读取**：支持读取邮件功能。
- **邮件删除**：支持删除邮件功能。

### 2.7 七牛云工具

- **七牛云配置**：支持七牛云配置管理。
- **七牛云工具**：支持七牛云存储操作。

### 2.8 许可证管理

- **硬件信息获取**：支持获取计算机硬件标识信息。
- **许可证生成**：支持生成基于硬件信息的软件许可证。
- **许可证验证**：支持验证和解析许可证信息。

### 2.9 API 响应封装

- **统一响应结果**：提供统一的 API 响应格式，包含状态码、消息和数据。
- **响应状态码**：定义了 HTTP 标准状态码和业务自定义状态码。
- **分页响应封装**：封装分页查询结果，包含总记录数、每页记录数、当前页码、总页数和数据列表。
- **便捷静态方法**：提供 `success()`、`fail()` 等静态方法，简化 API 响应的创建。
- **泛型支持**：支持泛型数据类型，适用于各种 API 返回数据场景。
- **支持分页查询**：集成分页功能，便于处理大量数据的分页查询结果。

## 3. 详细使用示例

### 3.1 地理编码服务使用示例

#### 3.1.1 高德地图反地理编码

```java
// 初始化反地理编码器
AmapRegeoCoder amapRegeoCoder = new AmapRegeoCoder.Builder("your_api_key").build();

// 执行反地理编码
AmapResponse response = amapRegeoCoder.reverseGeocode(116.397428, 39.90923);

// 解析响应结果
if (response.isSuccess()) {
    String formattedAddress = response.getFormattedAddress();
    System.out.println("格式化地址: " + formattedAddress);
    
    // 获取详细地址信息
    AddressComponent addressComponent = response.getAddressComponent();
    System.out.println("省份: " + addressComponent.getProvince());
    System.out.println("城市: " + addressComponent.getCity());
    System.out.println("区县: " + addressComponent.getDistrict());
    System.out.println("街道: " + addressComponent.getStreet());
    System.out.println("门牌号: " + addressComponent.getDoorplate());
}
```

#### 3.1.2 高德地图地理编码

```java
// 初始化地理编码器
AmapGeoCoder amapGeoCoder = new AmapGeoCoder.Builder("your_api_key").build();

// 执行地理编码
AmapGeoResponse response = amapGeoCoder.geoCode("北京市海淀区中关村");

// 解析响应结果
if (response.isSuccess() && response.getGeocodes() != null && !response.getGeocodes().isEmpty()) {
    Geocode geocode = response.getGeocodes().get(0);
    System.out.println("经度: " + geocode.getLongitude());
    System.out.println("纬度: " + geocode.getLatitude());
    System.out.println("格式化地址: " + geocode.getFormattedAddress());
}
```

#### 3.1.3 天地图地理编码

```java
// 初始化天地图地理编码器
TiandituGeocoder tiandituGeocoder = new TiandituGeocoder.Builder("your_api_key").build();

// 执行地理编码
TiandituResponse response = tiandituGeocoder.geocode("北京市海淀区中关村");

// 解析响应结果
if (response.isSuccess()) {
    System.out.println("经度: " + response.getLongitude());
    System.out.println("纬度: " + response.getLatitude());
    System.out.println("地址: " + response.getAddress());
}
```

### 3.2 坐标转换工具使用示例

#### 3.2.1 本地坐标转换

```java
// 创建坐标点
CoordinateConverter.Point wgs84Point = new CoordinateConverter.Point(116.397428, 39.90923); // 北京天安门

// WGS-84转GCJ-02
CoordinateConverter.Point gcj02Point = CoordinateConverter.wgs84ToGcj02(wgs84Point);
System.out.println("GCJ-02坐标: " + gcj02Point);

// GCJ-02转BD-09
CoordinateConverter.Point bd09Point = CoordinateConverter.gcj02ToBd09(gcj02Point);
System.out.println("BD-09坐标: " + bd09Point);

// BD-09转GCJ-02
CoordinateConverter.Point backToGcj02 = CoordinateConverter.bd09ToGcj02(bd09Point);
System.out.println("转回GCJ-02: " + backToGcj02);

// GCJ-02转WGS-84
CoordinateConverter.Point backToWgs84 = CoordinateConverter.gcj02ToWgs84(backToGcj02);
System.out.println("转回WGS-84: " + backToWgs84);
```

## 5. 依赖和安装

在项目的 `pom.xml` 的 `dependencies` 中加入以下内容：

<dependency>

    <artifactId>jk-tool-jdk8</artifactId>
    <version>0.0.4</version>
</dependency>

implementation 'io.github.jukejuke:jk-tool-jdk8:0.0.4'
